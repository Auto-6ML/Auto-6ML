# MLCGR

这是一个 PyTorch框架下载数据集CIFAR-10上的代码，用于研究利用元学习校正噪声标签。

## 运行

python main.py

主要超参数在 `config.py` 中修改，包括噪声率、噪声类型、训练轮数、学习率、特征约束权重等。


project/
├── data/           # 数据存放
├── losses/         # 特征约束正则项损失以及负学习损失
├── models/         # 提取特征用的resnet网络，分类使用的线性分类网络，校正标签的元网络
├── trainer/        # 数据划分和特征提取网络主网络和元网络的训练
├── config          # 配置文件，存放模型的主要超参数 （这里的默认超参数设置可以跑但是应该效果不是最好的）
├── utils/          # 工具函数
└── main            # 运行程序入口 （我是用pycharm本地跑的，这个main.py就是程序入口）
